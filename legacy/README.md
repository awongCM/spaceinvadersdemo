# Legacy Space Invaders

Classic Space Invaders game built with vanilla JavaScript, jQuery, and HTML5 Canvas.

## Development

To work on the legacy game, simply open `main.html` in your browser or serve it with any local server.

## Build

To create a production build:

```bash
npm run build
```

This will create a `dist/` folder with all the game files ready for deployment.

## Structure

- `main.html` - Game entry point
- `js/` - Game logic and sprites
- `css/` - Styling
- `images/` - Sprite sheets
- `media/` - Sound effects
